# Atari 2600 Customization

Este repositório contém os projetos desenvolvidos para a disciplina de Desenvolvimento de Jogos Clássicos, com foco em programação Assembly para o Atari 2600.

## Sobre o Projeto

- **Plataforma:** Atari 2600
- **Linguagem:** Assembly 6502
- **Ferramentas:** DASM (assembler), Stella (emulador)

## Estrutura do Repositório

- `src/` — Códigos-fonte dos jogos (ex: pong.asm, boxing.asm)
- `bin/` — Binários gerados para rodar no emulador
- `dasm/` — Montador DASM e includes
- `Stella/` — Emulador Stella
- `docs/` — Materiais de apoio e documentação

## Como montar e rodar

1. Edite o arquivo `.asm` desejado em `src/`.
2. Use a task "Montar com DASM" para gerar o `.bin` correspondente em `bin/`.
3. Use a task "Executar no Stella" para rodar o binário no emulador.

## Observações

- Todos os códigos são de autoria do aluno, podendo conter adaptações de exemplos clássicos.
- Comentários e organização seguem o padrão da disciplina.
